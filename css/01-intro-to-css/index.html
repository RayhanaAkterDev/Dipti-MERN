Introduction to CSS

CSS (Cascading Style Sheets) is used to control the appearance and layout of HTML elements. CSS can be applied in three different ways: inline CSS, internal CSS, and external CSS.

Inline CSS

This Heading Uses Inline CSS

Styles are applied directly through the style attribute.

Internal CSS

This Heading Uses Internal CSS

Styles are written inside a <style> element within the <head>.

External CSS

This Heading Uses External CSS

Styles are written in a separate .css file and linked to the HTML document.

CSS Selectors

CSS selectors are used to target HTML elements and apply styles to them. Different selectors allow developers to target elements based on their tag name, class, ID, or other criteria.

Element Selector

An element selector targets all occurrences of a specific HTML element. In this example, the selector targets every <p> element on the page.

Class Selector

A class selector targets elements that share the same class attribute value.

This element belongs to the highlight class.

ID Selector

An ID selector targets a single unique element using its id attribute.

This paragraph is targeted using the unique-text ID.

Group Selector

A group selector applies the same styles to multiple selectors at once.

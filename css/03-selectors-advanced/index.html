CSS Selectors (Advanced)

Advanced selectors allow developers to target elements based on their relationships with other elements or specific HTML attributes.

Descendant Selector

A descendant selector targets elements that are nested inside another element, regardless of depth.

This nested paragraph is also selected by the same selector.

Child Selector

A child selector targets only direct children of a parent element.

This paragraph is a direct child and is selected.

This paragraph is not a direct child and is not selected.

Adjacent Sibling Selector

An adjacent sibling selector targets the first element that immediately follows another element.

Heading

This paragraph is selected because it comes immediately after the heading.

This paragraph is not selected by the adjacent sibling selector.

General Sibling Selector

A general sibling selector targets all matching siblings that appear after a specified element.

Section Title

This div is not selected.

Attribute Selector

Attribute selectors target elements based on the presence or value of an HTML attribute.

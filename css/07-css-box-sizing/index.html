CSS Box Sizing

The box-sizing property determines how width and height are calculated for an element.

content-box (Default)

content-box

Width = 200px + padding + border

border-box

border-box

Width remains 200px

Side By Side Comparison

content-box
border-box

Both boxes have width: 200px, padding: 20px, and border: 5px.

The content-box element becomes larger, while border-box keeps its specified width.

Global Reset

* {
    box-sizing: border-box;
}

This is commonly used in modern projects to make sizing more predictable.

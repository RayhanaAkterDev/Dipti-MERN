CSS Float & Clear

The float property moves an element to the left or right, allowing surrounding content to wrap around it. The clear property prevents elements from wrapping around floated elements.

Modern CSS

Before Flexbox and Grid, float was commonly used for page layouts.

Today, float is mainly used for wrapping text around images, while Flexbox and Grid are preferred for layout design.

HTML Block and Inline Elements

HTML elements are generally categorized into block elements and inline elements based on how they are displayed in a web page.

Block Elements

Block elements always start on a new line and occupy the full available width by default.

Example Heading (Block Element)

This is the first paragraph.

This is the second paragraph.

Inline Elements

Inline elements only take up as much width as necessary and remain within the flow of surrounding text.

This paragraph contains strong text, emphasized text, highlighted text, and H2O, all of which are inline elements.

Key Difference

Block elements start on a new line and take up the full available width.

Inline elements remain within the same line and only occupy the width needed by their content.

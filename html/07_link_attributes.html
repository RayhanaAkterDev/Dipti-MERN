HTML Link Attributes

The <a> element supports several attributes that define the destination and behavior of a hyperlink.

href Attribute

The href attribute specifies the destination of the hyperlink.

Visit Example Website

target Attribute

The target attribute specifies where the linked document will open.

Open Example Website in a New Tab

rel Attribute

The rel attribute defines the relationship between the current page and the linked resource.

When using target="_blank", it is recommended to include rel="noopener noreferrer" for improved security and privacy.

title Attribute

The title attribute provides additional information about a link. It is commonly displayed as a tooltip when the user hovers over the link.

Hover over this link

The href attribute is required to create a working hyperlink.

When opening links in a new tab using target="_blank", include rel="noopener noreferrer" as a recommended modern practice.

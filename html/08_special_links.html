HTML Special Links

Besides navigating to web pages, the <a> element can also create links for sending emails, making phone calls, and navigating to specific sections within a web page.

Email Links

Use the mailto: URL scheme to open the user's default email application.

support@example.com

Telephone Links

Use the tel: URL scheme to initiate a phone call on supported devices.

+1 234 567 890

Anchor Links

Anchor links allow users to jump to another section within the same web page.

Go to Contact Section

Contact Section

This is the destination of the anchor link.

Back to Top

Email and telephone links improve user experience by allowing users to contact you directly from supported devices.

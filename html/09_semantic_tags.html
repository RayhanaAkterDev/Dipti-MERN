Use semantic elements whenever appropriate instead of relying only on generic elements like <div>. Semantic HTML improves code readability, maintainability, and search engine optimization (SEO).

HTML5 provides many more semantic elements than the ones demonstrated in this example.

Some commonly used semantic elements include <address>, <figure>, <figcaption>, <details>, <summary>, <time>, <mark>, and <dialog>.

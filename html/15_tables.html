HTML Tables

The <table> element is used to display data in rows and columns.

Basic Table Structure

A basic HTML table is built using the following elements: <table>, <caption>, <thead>, <tbody>, <tr>, <th>, and <td>.

Common Table Elements

  • The <caption> element provides a title for the table.
  • The <thead> element groups the table header.
  • The <tbody> element groups the table body.
  • The <tr> element defines a table row.
  • The <th> element defines a header cell.
  • The <td> element defines a data cell.

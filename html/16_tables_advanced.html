HTML Tables (Advanced)

HTML tables provide additional elements and attributes for creating more structured, accessible, and complex tables.

Advanced Table Elements and Attributes

Advanced HTML tables commonly use the following elements and attributes: <tfoot>, scope, colspan, and rowspan.

Common Advanced Table Elements

  • <tfoot> groups footer or summary information.
  • The scope attribute identifies row and column header cells.
  • The colspan attribute allows a cell to span multiple columns.
  • The rowspan attribute allows a cell to span multiple rows.

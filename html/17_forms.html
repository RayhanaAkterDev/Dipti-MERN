HTML Forms

The <form> element is used to collect user input and submit data to a server.

Basic Form Structure

A basic HTML form commonly uses the following elements: <form>, <fieldset>, <legend>, <label>, <input>, <textarea>, <select>, <option>, <optgroup>, <datalist>, and <button>.

Common Form Elements

  • <form> - creates the form container.
  • <fieldset> - groups related form controls.
  • <legend> - provides a title for a fieldset.
  • <label> - describes a form control.
  • <input> - creates an input field.
  • <textarea> - creates a multi-line text input.
  • <select> - creates a drop-down list.
  • <option> - defines an item inside a drop-down list.
  • <optgroup> - groups related options inside a drop-down list.
  • <datalist> - provides predefined suggestions for an input field.
  • <button> - creates a clickable button.

Common Form Attributes

  • action specifies where the form data is sent.
  • method specifies how the form data is sent, usually GET or POST.

Modern HTML Recommendation: Always associate a <label> with its corresponding form control using the for and id attributes. This improves accessibility and makes forms easier to use.

Common Input Attributes

  • placeholder displays a hint.
  • value sets a default value.
  • readonly prevents editing.
  • disabled disables the input.
  • autocomplete controls browser autofill.
  • autofocus focuses an input on page load.
  • checked preselects a checkbox or radio button.
  • multiple allows multiple file or email selections.
  • accept restricts upload file types.

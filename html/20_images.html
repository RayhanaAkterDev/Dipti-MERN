HTML Images

The <img> element is used to display images on a webpage.

Image Paths

Images can be loaded using either a relative path or an absolute URL.

  • Relative Path: Loads an image from the same website or project.
  • Absolute URL: Loads an image from another website using its complete web address.

Common Image Attributes

The following attributes are commonly used with the <img> element:

  • src - specifies the image location.
  • alt - provides alternative text if the image cannot be displayed.
  • width - specifies the image width.
  • height - specifies the image height.
  • title - displays additional information when the user hovers over the image.
  • loading="lazy" - delays loading images until they are needed, improving page performance.

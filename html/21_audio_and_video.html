HTML Audio & Video

HTML provides the <audio> and <video> elements for embedding multimedia content on a webpage.

Common Media Attributes

The following attributes are commonly used with both <audio> and <video>.

  • controls - displays the built-in media controls.
  • autoplay - starts playback automatically when the page loads.
  • loop - continuously repeats the media.
  • muted - starts playback with the sound turned off.
  • preload - specifies how much media should be loaded before playback.
  • <source> - allows multiple media files to be provided for better browser compatibility.

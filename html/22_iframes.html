HTML Embedded Media (iframe)

The <iframe> element is used to embed another webpage or external content inside the current webpage.

Common iframe Attributes

The following attributes are commonly used with the <iframe> element.

  • src - specifies the webpage or resource to embed.
  • width - specifies the iframe width.
  • height - specifies the iframe height.
  • title - provides a description for accessibility.
  • loading="lazy" - delays loading the iframe until it is close to the viewport.
  • allowfullscreen - allows supported content (such as videos) to enter fullscreen mode.

HTML Additional Features

HTML provides many additional elements and attributes that are useful in specific situations. While beginners may not use them every day, knowing they exist will help you as you continue learning web development.

1. HTML Comments

Comments are ignored by the browser and are mainly used to document code. 

<!-- This is a single-line comment -->

<!--
    This is a
    multi-line comment.
-->

4. Useful Global Attributes

  • id - Uniquely identifies an element.
  • class - Groups elements for CSS and JavaScript.
  • title - Provides additional information on hover.
  • hidden - Hides an element.
  • tabindex - Controls keyboard navigation order.
  • contenteditable - Makes content editable.
  • draggable - Allows an element to be dragged.
  • spellcheck - Enables or disables spell checking.
  • translate - Indicates whether content should be translated.

5. Learn Next

Some HTML features are closely related to CSS and JavaScript and are usually learned later:

  • data-* attributes → JavaScript DOM
  • <script> → JavaScript
  • <canvas> → JavaScript Graphics
  • SVG → HTML + CSS
  • Web Components → Advanced HTML & JavaScript
